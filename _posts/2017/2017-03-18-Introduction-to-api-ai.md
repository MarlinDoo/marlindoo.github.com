第一篇 [Wellcome - API.AI](https://docs.api.ai/docs/welcome)

API.AI是一个自然语言理解（Natural Language Understanding - NLU）平台，可以让开发者（以及非开发者）能够轻松将智能和复杂的用户会话接口设计和集成到移动应用、Web应用、设备、机器人中。

我们的目标是将创建和集成复杂的会话用户界面的过程尽可能简单。
